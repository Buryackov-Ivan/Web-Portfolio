Laboratory work 9

Создайте одномодульный Java-проект с использованием Maven и Spring Boot. Это консольное приложение, принимающее в аргументе командной строки название факультета, института или кафедры с сайта Электронный атлас .

Для конкретной кафедры извлеките URL из атрибута href HTML элемента --a class="alist" href="..."-- название кафедры --/a-- (-- ⮀ <>). Затем распарсьте страницу кафедры и выведите на экран или сохраните в CSV файл данные из таблицы преподавателей.

Если речь идёт о названии факультета или института, то надо в цикле распарсить всен страницы кафедр и получить данные о всех преподавателях.

Рекомендации:
⮚Используйте видеоматериалы курса.
⮚Установите WSL (Windows for Linux), Git Bash или Cygwin для эмуляции Bash Linux.
⮚Установите SDKMAN в Bash и используйте его для установки нужных версий Java (JDK), Maven и Spring Boot CLI.
⮚Генерируйте базовый проект с помощью команды spring init --build maven --force.
⮚Используйте JSoup для парсинга сайта, добавив его в зависимости в pom.xml.
⮚Настройте IntelliJ IDEA на установленные JDK и Maven.
⮚Опционально, установите библиотеки для SDKMAN, написанные на Rust. ★
